## Usage

To use this project, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Web4application/kubu-hai.git
    ```
2. Navigate to the project directory:
    ```bash
    cd kubu-hai
    ```
3. Run the main script:
